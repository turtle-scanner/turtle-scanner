# Stock Alpha Lab (Bonde Edition)💎

A powerful investment terminal and counseling study system designed for premium users.

## Features
- **🎯 MAGNA-PRO:** Momentum Burst (MB), Episodic Pivot (EP), and Anticipation scanner.
- **📈 Master Insights:** Strategies from Pradeep Bonde, William O'Neil, and Mark Minervini.
- **🕒 Dual Clock:** Live tracking of Korea (KST) and USA (ET) market times.
- **🔐 Secure Access:** Private terminal protected by an access code.
- **🐢 Counseling System:** Random case studies for counseling teacher preparation.

## Tech Stack
- **Backend:** Python, Streamlit
- **Data:** yfinance, pandas
- **Frontend:** HTML5, CSS3 (Vanilla), Chart.js
- **Styling:** Custom Black & Yellow Premium Theme

## How to Run
1. Install dependencies:
   ```bash
   pip install streamlit pandas yfinance pytz
   ```
2. Start the terminal:
   ```bash
   streamlit run terminal_app.py
   ```
